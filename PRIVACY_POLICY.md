Privacy Policy for OmniGuard
Last Updated: April 20, 2026

1. No Data Collection
OmniGuard does NOT collect, store, or transmit any personal information. All data stays on your device.

2. Permissions Used
QUERY_ALL_PACKAGES

Use: Display all installed apps

Data: App names, icons, package names

Storage: Local database only

PACKAGE_USAGE_STATS

Use: Find unused apps

Data: Last usage timestamps

Storage: Local database only

READ_EXTERNAL_STORAGE

Use: Analyze storage usage

Data: File types and sizes

Note: Cannot read file contents, only metadata

POST_NOTIFICATIONS

Use: Security alerts

Data: None - only local notifications

3. Third-Party Services
OmniGuard uses NO third-party analytics, crash reporting, or ad networks.

4. Data Deletion
Uninstalling OmniGuard removes all data automatically.
